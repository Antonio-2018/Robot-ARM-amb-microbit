# Robot-ARM-amb-microbit

Projecte de desenvolupament d'un braç robòtic ARM. 

- Versió amb peces impreses en 3D.
- Moviments controlats amb els botons i logo de la targeta micro:bit.
- Seqüències automàtiques.
- Dossier pas a pas del muntatge, calibració i programació bàsica.
- Fitxers .hex per Makecode.
